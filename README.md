                                                                                                                                                                             Smart-Match (2025)
מערכת Full Stack להתאמת מועמדים למשרות הייטק באמצעות אלגוריתם הונגרי.  
כוללת ממשק מועמדים, טופס קורות חיים עם דרישות וכישורים, ומערכת ניהול משרות למנהלים.  
פיתוח צד לקוח ב־React + Redux עם ולידציות מתקדמות.  
צד שרת ב־ASP.NET Core עם Entity Framework ו־SQL Server.
